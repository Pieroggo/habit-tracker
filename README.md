# Habit Tracker

## Prosta aplikacja do zarządzania nawykami, wspomagająca wprowadzanie nowych dobrych nawyków i kontynuowanie ich 

Funkcjonalności:
- Zarządzanie nawykami, czynnościami, celami
- Odhaczenie nawyków/celi, bądź aktualizacja postępów
- Zapis historii wykonań celi/nawyków
- Streak w utrzymaniu danych nawyków
- Statystyki (procent utrzymania nawyku, całkowita ilość odhaczeń, ulubiony nawyk)

Technologie:
- Java 25
- Spring
- PostgreSQL
- Docker
- Maven
- Hibernate
